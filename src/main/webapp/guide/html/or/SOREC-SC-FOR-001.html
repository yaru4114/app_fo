지정가 주문이란?

고객이 원하는 가격과 수량을 지정하여 주문하는 방식입니다.
접수부터 주문 체결까지 ‘당일 거래(개장 하절기 09시/동절기 10시 ~ 폐장17시)’를 원칙으로 하며,
지정한 가격에 도달 시 자동으로 주문 체결까지 완료됩니다.
(입력한 지정가에 도달하지 않을 경우, 주문이 체결되지 않습니다.)

  • 지정가

지정가 주문

마음속에 원하는 가격과 수량이 있으세요?
원하는 가격과 수량 설정으로 만족스러운
주문체결까지 시작해보세요.

내가 원하는 가격과 수량을 콕 집어서 주문 안되나?
항상 LIVE로만 구매 해야 하나???

서린닷컴

사장님~이제 서린닷컴에서 지정가 주문이 가능해졌습니다.

A기업

그래요? LIVE 주문 뿐만 아니라
내가 원하는 가격 수량을 지정해서도 주문이 가능하다구요?

서린닷컴

네 그렇습니다. 생각하셨던 가격과 수량이 있으셨다면
이제! 지정가 주문 방식을 이용해보세요.
단, 지정가 단어 뜻처럼 입력한 가격에 도달 했을 경우
자동 체결이 된다는 사실, 아시죠?

A기업

좀 더 자세히 지정가 체결 방식과 운영 시간 정보 알려주세요.

서린닷컴

네. 지정가 주문은 원하는 가격과 수량을
당일 기준, 서린닷컴 운영 시간 내에 지정 주문하고
체결 여부 까지 진행되는 주문 방식입니다.
(하절기 : 09:00~17:00 / 동절기 : 10:00~17:00)

A기업

부분 체결은 뭔가요?

서린닷컴

가격 도달 시, 지정한 수량이 구매 가능 수량과 일치하지 않더라도
부분 체결을 통해 일부 수량을 확보하는 주문 요청입니다.

A기업

혹시 지정가 주문하고 나서 수정도 가능한가요?

서린닷컴

네, 자동 체결 전까지는 수정 또는 취소가 가능합니다.
수정은 가격, 수량 모두 가능합니다. 수량 수정은 감량만 가능하며
가격 수정 시에는 체결 순위가 뒤로 밀려난다는 사실 기억하세요.

A기업

주문 방식이 다양화되니, 이용에 도움이 되겠네요~

서린닷컴

비철금속을 주문하는 고객님의 다양한 의견을 존중합니다.
현장에서의 주문 방식을 최대한 고려하여 제공하도록 계속 노력하겠습니다.

지정가 체결 순위 안내

당일거래 하절기 09시~17시 / 동절기 10시~17시

  1. 0 재고

    선택한 상품의 재고가 있을 경우 체결됩니다. (지정 가격이 도달하기 전 해당 상품의 재고가 소진되면 미체결로 전환됩니다.)

  2. 1 가격

    지정 가격 도달 시, 최우선 체결됩니다.

  3. 2 시간

    동일 가격 시, 접수 시간 순서로 체결됩니다.

  4. 3 수량

    가격과 시간이 동일 시에는 주문 수량이 큰 순서로 체결됩니다.

지정가 이용 방법

  • 01

    가격 주문 방식 : 지정가를 선택하세요.

  • 02

    지정가를 입력 설정하세요.

  • 03

    호가창에 내가 입력한 가격대에
    내 체결 순서를 안내합니다.

  • 04

    주문하면, 마이페이지 > 지정가 주문 목록에서
    확인 가능합니다.

  • 05

    체결 전까지는 수정 또는 접수 취소도 가능합니다.
